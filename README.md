# Gestion-de-RRHH
Este proyecto es un sistema de gestión de recursos humanos desarrollado en Java con SQLite, que permite administrar información clave de los empleados como asistencias, horas trabajadas, préstamos, estado laboral (activos e inactivos) y cálculo automático de la nómina, determinando el salario neto de cada trabajador.

El proyecto está estructurado bajo un enfoque tipo CRUD, organizado en diferentes capas: conexión a la base de datos, modelos (gestión de datos), vistas (interfaz gráfica) y lógica del sistema. Esto permite una administración eficiente, modular y escalable.

Dentro de la carpeta del proyecto se incluyen todas las clases necesarias para el funcionamiento del sistema, la base de datos SQLite donde se almacena la información, y el archivo ejecutable que permite correr la aplicación de manera directa. Esta organización facilita tanto el uso como la distribución del sistema.
